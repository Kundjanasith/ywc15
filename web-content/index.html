Young Web Master Camp

"ค่ายสร้างเว็บหนึ่งเดียวของไทย ที่กำลังตามหาคนทำเว็บสายเลือดใหม่ มาร่วมสร้าง ”เว็บไซต์นวัตกรรม นำประเทศ” กับพวกเรา และเพื่อนๆ จากหลากหลายสาขา ไม่ว่าจะเป็น Content,

Design, Marketing และ Programming ที่ต้องร่วมกันฟันฝ่าภารกิจ แถมยังได้ใกล้ชิดกูรูจากวงการเว็บไทย ที่จะมาแนะแนวทาง พร้อมให้คำวิจารณ์อย่างดุ เด็ด เผ็ด มัน หากคุณคือนักศึกษา

SEMI_FINAL ROUND

การสัมภาษณ์จะจัดขึ้นในวันที่ 26 พฤศจิกายน 2560 ณ อาคาร ซี.พี.ทาวเวอร์ 1 (สีลม) ซึ่งจะแบ่งออกเป็น 2 รอบ คือ รอบช่วงเช้าตั้งแต่เวลา 9.00 น. ถึง 12.00 น. และ รอบช่วงบ่ายตั้งแต่เวลา 13.00 น. ถึง 18.00 น.

สิ่งที่ต้องเตรียมมาในวันสัมภาษณ์

     1. บัตรประชาชนสำหรับการแลกบัตรเข้าอาคาร ซี.พี.ทาวเวอร์ 1 (สีลม) และ บัตรนักศึกษาสำหรับการลงทะเบียนสัมภาษณ์ กรุณาแต่งกายด้วยชุดนักศึกษา

     2. การบ้านและสิ่งที่กรรมการสาขากำหนดไว้ กรุณาอ่านรายละเอียดการบ้านและสิ่งที่กรรมการให้เตรียมมาให้ครบถ้วน หากสาขาใดต้องใช้โน้ตบุ๊ค ควรชาร์ตแบตเตอรี่และเตรียมอินเทอร์เน็ตส่วนตัวมาให้พร้อม เนื่องจากสถานที่ไม่มีบริการอินเทอร์เน็ตให้ใช้

     3. Portfolio สามารถนำมาประกอบการสัมภาษณ์ได้ สำหรับน้อง ๆ สาขาดีไซน์จะต้องนำ Portfolio มาด้วยทุกคน

การเดินทางมาสัมภาษณ์

     1. ด้วยรถไฟฟ้า BTS สามารถลงสถานีศาลาแดง ณ ทางออกที่ 2

     2. ด้วยรถไฟฟ้า MRT สามารถลงสถานีสีลม ณ ทางออกที่ 2 โดยเดินเรียบทางเท้าไปตามถนนสีลม

     3. ด้วยรถประจำทาง สามารถขึ้นใช้บริการสาย 15, 77, 155, 504, 177, 76

หมายเหตุ:

     สำหรับน้อง ๆ ที่ไม่สะดวกเดินทางมาสัมภาษณ์ที่อาคาร CP Tower สีลม ให้ Inbox มาทางเพจเฟสบุ๊ค Young Webmaster Camp ภายในวันที่ 20 พฤศจิกายน 2560

การบ้านสาขาคอนเทนท์

     ให้น้อง ๆ เลือกทำคอนเทนต์ใด คอนเทนต์หนึ่งจาก 2 หัวข้อด้านล่าง โดยต้องทำเป็นคอนเทนต์ออกมาจริงเท่านั้น ไม่จำกัดรูปแบบการนำเสนอ
     หัวข้อที่ให้เลือก
     1. คอนเทนต์ที่ทำให้ผู้ใช้ Social เข้าใจ Cyberbullying และตระหนักถึงปัญหาที่เกิดขึ้นจาก Cyberbullying
     2. คอนเทนต์ที่ทำให้คนไทยเปลี่ยนพฤติกรรมเสพติดหน้าจอ เสพติด Social หันมาพูดคุยกัน เจอหน้ากันในชีวิตจริง ๆ มากกว่าที่เป็นอยู่

การบ้านสาขาดีไซน์

     ให้ผู้ผ่านเข้ารอบสัมภาษณ์ออกแบบเว็บไซต์ของ "คณะรักษาความสงบแห่งชาติ หรือ คสช." เพื่อเป็นสื่อกลางในการประชาสัมพันธ์นโยบายต่าง ๆ และเป็นสื่อกลางระหว่างประชาชนกับคณะรักษาความสงบแห่งชาติ โดยให้ออกแบบในลักษณะของ One Page Design Website
     ** สำหรับผู้ผ่านเข้ารอบสัมภาษณ์คนใดทำ CSS มา คณะกรรมการจะพิจารณาเป็นพิเศษ

การบ้านสาขามาร์เก็ตติ้ง

     ให้ทำแผนการตลาด (Marketing Plan) สำหรับเว็บไซต์หนึ่งเว็บไซต์ โดยเลือกเว็บไซต์ใด ๆ ก็ได้ ทั้งเว็บไซต์ของไทยและเว็บไซต์ของต่างประเทศ
     นำเสนอในรูปแบบพรีเซนเทชั่น ไม่เกิน 10 หน้า ภายในเวลา 5 นาที

การบ้านสาขาโปรแกรมมิ่ง

ให้เขียน เว็บไซต์ประกาศผลผู้ผ่านเข้ารอบสัมภาษณ์ของ YWC#15 โดยใช้ข้อมูลจาก API โดยมี Feature ดังนี้
     • ดึงข้อมูลจาก API โดยตรง
     • ให้แสดงผลแยกแต่ละสาขา
     • มีระบบค้นหาชื่อผ่านกล่อง Search
     • ความสามารถหรือ Feature พิเศษอื่น ๆ ที่มีความแตกต่าง และแสดงความสามารถของน้องออกมาให้ได้มากที่สุด
เมื่อทำเสร็จแล้วให้ Push Source Code ขึ้น GitHub ก่อนเวลาสัมภาษณ์

Questions

พี่ๆ ค่าย #YWC15 จะประกาศผลเมื่อไหร่ แล้วจะรู้ได้ยังไงว่า ผ่านการคัดเลือกแล้ว ?

ค่าย YWC รับสมัครใครบ้าง ?

 

Answer

คำถาม : เขียนโค้ดไม่ได้ ทำเว็บไม่เป็น สมัครได้ไหม ?

 

คำตอบ : ได้แน่นอนค่ะ เพราะเรารู้ว่าแต่ละคนมีความถนัดไม่เหมือนกัน แต่สิ่งที่จะทำให้ทุกคนฝ่าฟันภารกิจไปได้ ต้องเกิดจากการร่วมมือกันทั้ง 4 ด้าน ถ้าเป็นคนชอบคิด ชอบเขียน ชอบเล่าเรื่อง ที่ทำให้คนที่ได้อ่าน ได้ฟังต้องตะลึง แนะนำสาขา Content ส่วนคนที่ว่างเมื่อไหร่เป็นต้องหยิบจับดินสอปากกามาขีดเขียน วาดนู่น ออกแบบนี่ แนะนำสาขา Design สำหรับคนที่มีวิสัยทัศน์กว้างไกล ชอบจัดการและวางแผน เพื่อให้บรรลุวัตถุประสงค์ แนะนำสาขา Marketing หรือถ้าเป็นคนที่มีพื้นฐานการเขียนเว็บในระดับหนึ่ง สาขา Programming ก็ไม่ได้ยากเกินความสามารถ แค่มีความตั้งใจ และใช้ทักษะพื้นฐาน และความถนัดที่มี แค่นี้ก็สามารถสมัครได้แล้วจ้า
